# no-emo 

使用 React Native 和 Expo 的 App

## How to Use

1. Git clone to your local

```
git clone https://github.com/Hubert0916/no-emo.git
```

2. Enter "no-emo" directory


3. Install dependencies
```
npm install
```

4. Start Expo server
```
npx expo start
```

5. Scan QR-code on your mobile device. Done.
